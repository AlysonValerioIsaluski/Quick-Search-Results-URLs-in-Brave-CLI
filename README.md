# Quick Search Results URLs in Brave CLI
CLI java application that searches the web using the provided text as input using Brave search engine and prints out all relevant urls from the first page. Can be used for quick search results, without ever needing to see any bloat or having to wait for the page to load.

JDK 21 required. No external libraries or frameworks needed.

While in the directory where QuickBrowse.java is, in the terminal:

COMPILATION:
```javac QuickBrowse.java```

EXECUTION:
```java QuickBrowse insert-search-here```


EXAMPLES:
```java QuickBrowse chocolate cake recipe```

```java QuickBrowse "how to center a div"```
